Williamson TC1: Advection of Cosine Bell Verified

Alphas4
Fields5
Key Days0, 3, 6, 9, 12
Error Plots4

Definition

Passive advection of a compact cosine bell by solid-body rotation on the sphere. The rotation axis can be tilted relative to the latitude-longitude grid, so the bell can cross the coordinate poles.

What This Test Measures

This is the clean transport test: conservation, phase accuracy, numerical diffusion, overshoots/undershoots, and the ability to move a compact structure through polar coordinate singularities without distortion.

Expected Output

After one 12-day revolution the bell should return to its starting location. The exact and MITgcm contours should nearly overlap, the tracer integral should stay flat, and signed-error contours should remain small and symmetric rather than growing long noisy tails.

Initial Conditions And Equations

The tracer is a bell centered at (λc,θc)=(3π/2,0) with compact radius R=a/3:

\[q=\frac{q_0}{2}\left[1+\cos\left(\frac{\pi r}{R}\right)\right],\quad r
\[q=0,\quad r\ge R\]
\[\frac{Dq}{Dt}=\frac{\partial q}{\partial t}+\mathbf{v}\cdot\nabla_s q=0\]

The prescribed velocity is the TC2 solid-body field, but momentum is not stepped in this local MITgcm case.

Parameters

  • a=6.371e6 m, g=9.81 m s^-2, Ω=7.292e-5 s^-1 in the local runs.
  • q0=1000, R=a/3, U0=2πa/(12 days).
  • Rendered tilt angles: α=0, 0.05, 1.52, 1.57 rad.

Williamson TC2: Global Steady-State Nonlinear Zonal Geostrophic Flow Issues

Alphas4
Fields5
Key Days0, 3, 6, 9, 12
Error Plots4

Definition

A full nonlinear shallow-water steady state: global solid-body zonal flow balanced by the free-surface height field. The flow is exact for every time when the continuous balance is preserved.

What This Test Measures

Balanced nonlinear dynamics, metric terms, Coriolis terms, pole treatment, and preservation of a known steady state. The paper also identifies TC2 as the performance benchmark, typically using α=π/4 for a 5-day run.

Expected Output

A good run stays almost motionless in the rotating balance: height and velocity errors should remain near discretization error, with little gravity-wave noise. Mesh refinement should reduce the 5-day error norms cleanly.

Initial Conditions And Equations

The active Williamson cases are run as one-layer rotating shallow-water problems. The MITgcm fields reduce to horizontal velocity u=(u,v) and free surface η over depth H.

\[\frac{D\mathbf{u}}{Dt}+f\hat{k}\times\mathbf{u}=-g\nabla_s\eta+\mathcal{D}_{\mathbf{u}}+\mathbf{F}\]
\[\frac{\partial \eta}{\partial t}+\nabla_s\cdot[(H+\eta)\mathbf{u}]=S_{\eta}\]
\[\frac{Dq}{Dt}=\mathcal{D}_{q}+S_q \quad \text{when passive tracers are enabled}\]

Each testcase changes the analytic initial state, forcing, bathymetry, or expected diagnostic behavior of this same system.

With longitude λ, latitude θ, and tilt α:

\[u=U_0\left(\cos\theta\cos\alpha+\cos\lambda\sin\theta\sin\alpha\right)\]
\[v=-U_0\sin\lambda\sin\alpha\]
\[\mu=-\cos\lambda\cos\theta\sin\alpha+\sin\theta\cos\alpha\]
\[g h=g h_0-\left(a\Omega U_0+\frac{1}{2}U_0^2\right)\mu^2\]

Parameters

  • U0=2πa/(12 days).
  • g h0=2.94e4 m^2 s^-2, so local H0=g h0/g=2996.94 m.
  • Standard tilts: α=0, 0.05, π/2-0.05, π/2; local outputs use 0, 0.05, 1.52, 1.57 rad.

Williamson TC3: Steady Geostrophic Flow with Compact Support Issues

Alphas2
Fields5
Key Days0, 3, 6, 9, 12
Error Plots2

Definition

A steady nonlinear geostrophic flow like TC2, but the wind is a smooth compact jet. The active balanced flow occupies only a limited latitudinal band.

What This Test Measures

Whether a model can maintain a localized balanced flow without leakage, phase drift, ringing near smooth compact-support edges, or spurious gravity waves.

Expected Output

The compact jet and free-surface anomaly should stay steady. Errors should be localized near the jet edges if they appear at all, and conservation diagnostics should show little long-term drift.

Initial Conditions And Equations

The active Williamson cases are run as one-layer rotating shallow-water problems. The MITgcm fields reduce to horizontal velocity u=(u,v) and free surface η over depth H.

\[\frac{D\mathbf{u}}{Dt}+f\hat{k}\times\mathbf{u}=-g\nabla_s\eta+\mathcal{D}_{\mathbf{u}}+\mathbf{F}\]
\[\frac{\partial \eta}{\partial t}+\nabla_s\cdot[(H+\eta)\mathbf{u}]=S_{\eta}\]
\[\frac{Dq}{Dt}=\mathcal{D}_{q}+S_q \quad \text{when passive tracers are enabled}\]

Each testcase changes the analytic initial state, forcing, bathymetry, or expected diagnostic behavior of this same system.

The local implementation builds the velocity from a compact bump in rotated latitude θ' and computes the height by integrating the nonlinear geostrophic balance:

\[u'=U_0 B(\theta'),\quad v'=0\]
\[B(\theta')=0\quad \text{outside }[-\pi/6,\pi/2]\]
\[h=H_0+\int \text{balanced pressure-gradient contribution}\,d\theta'\]

Parameters

  • U0=2πa/(12 days), H0=2996.94 m.
  • Compact support interval: θ0=-π/6 to θ1=π/2.
  • Runs can be rotated by α to move the jet away from grid lines.

Scaffolded pending the validated TC4 analytic forcing hook.

Definition

A forced nonlinear shallow-water exact-solution test with a translating low. The prescribed forcing makes the unsteady analytic disturbance the target solution.

What This Test Measures

When complete, TC4 targets unsteady nonlinear accuracy with a known forced solution: advection of a strong localized low, source-term balance, and error growth in a moving feature.

Expected Output

A validated TC4 should track the translating low against the analytic solution. For the current scaffold, expect the page to document setup status rather than claim a reference-quality TC4 result.

Initial Conditions And Equations

The active Williamson cases are run as one-layer rotating shallow-water problems. The MITgcm fields reduce to horizontal velocity u=(u,v) and free surface η over depth H.

\[\frac{D\mathbf{u}}{Dt}+f\hat{k}\times\mathbf{u}=-g\nabla_s\eta+\mathcal{D}_{\mathbf{u}}+\mathbf{F}\]
\[\frac{\partial \eta}{\partial t}+\nabla_s\cdot[(H+\eta)\mathbf{u}]=S_{\eta}\]
\[\frac{Dq}{Dt}=\mathcal{D}_{q}+S_q \quad \text{when passive tracers are enabled}\]

Each testcase changes the analytic initial state, forcing, bathymetry, or expected diagnostic behavior of this same system.

The paper's TC4 adds analytic forcing terms to the mass and momentum equations so a localized low translates without changing form. In this repository the TC4 directory is intentionally marked as a scaffold until those forcing hooks are implemented in MITgcm local code.

Parameters

Williamson TC5: Zonal Flow Over an Isolated Mountain Pending validation

Alphas1
Fields5
Key Days0, 5, 10, 15
Error Plots0

Definition

Zonal shallow-water flow impinging on an isolated conical mountain. Unlike TC2 and TC3, this is not an exact steady-state comparison; topography actively generates waves.

What This Test Measures

Topographic source-term handling, conservation over variable bathymetry, wave generation, and whether the discretization creates unphysical noise near steep terrain.

Expected Output

The mountain should produce a coherent downstream wave response. There is no simple analytic truth field, so evaluate against reference solutions, global invariants, smoothness, and absence of terrain-locked artifacts.

Initial Conditions And Equations

The active Williamson cases are run as one-layer rotating shallow-water problems. The MITgcm fields reduce to horizontal velocity u=(u,v) and free surface η over depth H.

\[\frac{D\mathbf{u}}{Dt}+f\hat{k}\times\mathbf{u}=-g\nabla_s\eta+\mathcal{D}_{\mathbf{u}}+\mathbf{F}\]
\[\frac{\partial \eta}{\partial t}+\nabla_s\cdot[(H+\eta)\mathbf{u}]=S_{\eta}\]
\[\frac{Dq}{Dt}=\mathcal{D}_{q}+S_q \quad \text{when passive tracers are enabled}\]

Each testcase changes the analytic initial state, forcing, bathymetry, or expected diagnostic behavior of this same system.

The initialized flow is zonal and geostrophically adjusted around mountain topography:

\[u=U_0\cos\theta,\quad v=0\]
\[h_s=h_{s0}\left(1-\frac{r}{R}\right),\quad r
\[h_s=0,\quad r\ge R\]
\[b=-(H_0-h_s)\]

Parameters

  • U0=20 m s^-1, H0=5960 m.
  • Mountain height h_s0=2000 m, radius R=π/9.
  • Mountain center (λc,θc)=(3π/2,π/6).

Williamson TC6: Rossby-Haurwitz Wave Pending validation

Alphas1
Fields5
Key Days0, 7, 14
Error Plots0

Definition

A Rossby-Haurwitz wave initialized from a nondivergent planetary wave pattern. The shallow-water version is a demanding nonlinear wave propagation benchmark.

What This Test Measures

Longer-range nonlinear wave propagation, phase and amplitude errors, pole behavior, and conservation of geopotential, energy, vorticity, divergence, and potential enstrophy.

Expected Output

The wave should retain a coherent planetary-scale structure with controlled phase drift. Look for smooth contour evolution and slow, explainable drift in global invariants rather than noisy small-scale breakdown.

Initial Conditions And Equations

The active Williamson cases are run as one-layer rotating shallow-water problems. The MITgcm fields reduce to horizontal velocity u=(u,v) and free surface η over depth H.

\[\frac{D\mathbf{u}}{Dt}+f\hat{k}\times\mathbf{u}=-g\nabla_s\eta+\mathcal{D}_{\mathbf{u}}+\mathbf{F}\]
\[\frac{\partial \eta}{\partial t}+\nabla_s\cdot[(H+\eta)\mathbf{u}]=S_{\eta}\]
\[\frac{Dq}{Dt}=\mathcal{D}_{q}+S_q \quad \text{when passive tracers are enabled}\]

Each testcase changes the analytic initial state, forcing, bathymetry, or expected diagnostic behavior of this same system.

The local fields follow the standard wave-number-4 streamfunction/height form:

\[r=4\]
\[u=a\omega\cos\theta+aK\cos^{r-1}\theta\left(r\sin^2\theta-\cos^2\theta\right)\cos(r\lambda)\]
\[v=-aKr\cos^{r-1}\theta\sin\theta\sin(r\lambda)\]
\[h=H_0+\frac{a^2}{g}\left[A(\theta)+B(\theta)\cos(r\lambda)+C(\theta)\cos(2r\lambda)\right]\]

Parameters

  • r=4, K=7.848e-6 s^-1, ω=7.848e-6 s^-1.
  • H0=8000 m, flat bathymetry.
  • Diagnostics usually emphasize days 0, 7, and 14 plus global invariants.

Williamson TC7: Analyzed 500 mb Initial State Needs data

Requires a static analyzed-height and wind initial-condition file.

Main Panels0
PrimaryLinked media
EquationsShown
Expected OutputIncluded

Definition

Realistic analyzed 500 mb height and wind states used as shallow-water initial conditions. This test gives the suite atmospheric texture rather than another exact solution.

What This Test Measures

Robustness on realistic large-scale atmospheric states: polar flow, blocking-like patterns, strong zonal flow, and sensitivity of a numerical method to the character of the initial state.

Expected Output

The shallow-water model is not expected to reproduce the real atmosphere perfectly. Use high-resolution reference comparisons, day-1/day-5 forecast-error plots, and the same global invariants as TC6 to judge behavior.

Initial Conditions And Equations

The active Williamson cases are run as one-layer rotating shallow-water problems. The MITgcm fields reduce to horizontal velocity u=(u,v) and free surface η over depth H.

\[\frac{D\mathbf{u}}{Dt}+f\hat{k}\times\mathbf{u}=-g\nabla_s\eta+\mathcal{D}_{\mathbf{u}}+\mathbf{F}\]
\[\frac{\partial \eta}{\partial t}+\nabla_s\cdot[(H+\eta)\mathbf{u}]=S_{\eta}\]
\[\frac{Dq}{Dt}=\mathcal{D}_{q}+S_q \quad \text{when passive tracers are enabled}\]

Each testcase changes the analytic initial state, forcing, bathymetry, or expected diagnostic behavior of this same system.

The paper uses analyzed atmospheric states truncated to T42 spectral resolution, with optional nonlinear normal-mode initialization. This repository expects input/raw/tc7_initial_conditions.npz containing eta_m, u_m_s, v_m_s, and optionally bathymetry.

Parameters

  • Local reference depth H0=8000 m.
  • Paper states include 0000 GMT 21 Dec 1978, 16 Jan 1979, and 9 Jan 1979 examples.
  • Local grid: 1440 x 720 at 0.25 deg.
