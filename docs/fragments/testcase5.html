Definition

Zonal shallow-water flow impinging on an isolated conical mountain. Unlike TC2 and TC3, this is not an exact steady-state comparison; topography actively generates waves.

What This Test Measures

Topographic source-term handling, conservation over variable bathymetry, wave generation, and whether the discretization creates unphysical noise near steep terrain.

Expected Output

The mountain should produce a coherent downstream wave response. There is no simple analytic truth field, so evaluate against reference solutions, global invariants, smoothness, and absence of terrain-locked artifacts.

Initial Conditions And Equations

The active Williamson cases are run as one-layer rotating shallow-water problems. The MITgcm fields reduce to horizontal velocity u=(u,v) and free surface η over depth H.

\[\frac{D\mathbf{u}}{Dt}+f\hat{k}\times\mathbf{u}=-g\nabla_s\eta+\mathcal{D}_{\mathbf{u}}+\mathbf{F}\]
\[\frac{\partial \eta}{\partial t}+\nabla_s\cdot[(H+\eta)\mathbf{u}]=S_{\eta}\]
\[\frac{Dq}{Dt}=\mathcal{D}_{q}+S_q \quad \text{when passive tracers are enabled}\]

Each testcase changes the analytic initial state, forcing, bathymetry, or expected diagnostic behavior of this same system.

The initialized flow is zonal and geostrophically adjusted around mountain topography:

\[u=U_0\cos\theta,\quad v=0\]
\[h_s=h_{s0}\left(1-\frac{r}{R}\right),\quad r
\[h_s=0,\quad r\ge R\]
\[b=-(H_0-h_s)\]

Parameters

  • U0=20 m s^-1, H0=5960 m.
  • Mountain height h_s0=2000 m, radius R=π/9.
  • Mountain center (λc,θc)=(3π/2,π/6).
