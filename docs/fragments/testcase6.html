Definition

A Rossby-Haurwitz wave initialized from a nondivergent planetary wave pattern. The shallow-water version is a demanding nonlinear wave propagation benchmark.

What This Test Measures

Longer-range nonlinear wave propagation, phase and amplitude errors, pole behavior, and conservation of geopotential, energy, vorticity, divergence, and potential enstrophy.

Expected Output

The wave should retain a coherent planetary-scale structure with controlled phase drift. Look for smooth contour evolution and slow, explainable drift in global invariants rather than noisy small-scale breakdown.

Initial Conditions And Equations

The active Williamson cases are run as one-layer rotating shallow-water problems. The MITgcm fields reduce to horizontal velocity u=(u,v) and free surface η over depth H.

\[\frac{D\mathbf{u}}{Dt}+f\hat{k}\times\mathbf{u}=-g\nabla_s\eta+\mathcal{D}_{\mathbf{u}}+\mathbf{F}\]
\[\frac{\partial \eta}{\partial t}+\nabla_s\cdot[(H+\eta)\mathbf{u}]=S_{\eta}\]
\[\frac{Dq}{Dt}=\mathcal{D}_{q}+S_q \quad \text{when passive tracers are enabled}\]

Each testcase changes the analytic initial state, forcing, bathymetry, or expected diagnostic behavior of this same system.

The local fields follow the standard wave-number-4 streamfunction/height form:

\[r=4\]
\[u=a\omega\cos\theta+aK\cos^{r-1}\theta\left(r\sin^2\theta-\cos^2\theta\right)\cos(r\lambda)\]
\[v=-aKr\cos^{r-1}\theta\sin\theta\sin(r\lambda)\]
\[h=H_0+\frac{a^2}{g}\left[A(\theta)+B(\theta)\cos(r\lambda)+C(\theta)\cos(2r\lambda)\right]\]

Parameters

  • r=4, K=7.848e-6 s^-1, ω=7.848e-6 s^-1.
  • H0=8000 m, flat bathymetry.
  • Snapshots include the common key days 0, 3, 6, 9, 10, and 12, plus day 14 when available.
